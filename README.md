## 🛠️ Troubleshooting

### The file will not open
- Download the file again
- Make sure the download finished
- Check that Windows did not block it
- Try opening it as admin if your system requires it

### The page stays blank
- Refresh the browser
- Clear the browser cache for the app page
- Try a different browser
- Check that the app is still running

### Agents do not start
- Check your settings
- Make sure the model or provider details are saved
- Restart the app
- Try a new task with the default settings

### The app closes right away
- Reopen it from the original file
- Check for Windows security prompts
- Make sure your machine meets basic memory needs
- Restart Windows and try again
